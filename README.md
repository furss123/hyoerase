# HyoErase (지우개)

교실/공용 PC용 **초강력 정리·관리 도구**. 게임·VPN·스토어 블로트웨어를
철저히 지우고, 딥 시스템 청소·재설치 차단·자동 스케줄링까지 처리합니다.
오피스·문서·그림 등 학교 관련 파일은 절대 건드리지 않습니다.

`HyoErase (지우개) v1.3.0 | © 2026 HyoT. All rights reserved. | hyot.dev`

## 실행 방법
1. **`HyoErase 실행.cmd`** 더블클릭 → 관리자 권한 요청(예 클릭).
2. 창이 뜨면 프로그램·스토어 앱·시스템 캐시·포터블 실행파일을 자동 검사합니다.
3. 지울 항목을 확인하고 **🧹 선택 항목 싹 정리** → 확인창에서 **예**.

> 설치 불필요. Windows 10/11 기본 PowerShell로 실행됩니다.
> 딥 청소·강력 제거·잠금 기능에는 관리자 권한이 필요해 자동으로 요청합니다.

### 강력 제거 옵션
"제거 후 잔여 파일·폴더·바로가기까지 삭제"(기본 켜짐):
- 제거 전 **실행 중인 프로세스를 강제 종료**해 성공률을 높입니다.
- 정식 제거 후 남은 **설치 폴더·시작메뉴/바탕화면 바로가기**까지 정리합니다.
- 스토어 앱은 제거 후 **재설치 방지(프로비저닝 제거)**까지 처리합니다.

### ⏱ 자동 정리 예약
상단 **"⏱ 자동 정리 예약"** 버튼으로 Windows 작업 스케줄러에 등록하면,
**매일 저녁 7시** 게임·VPN·스토어앱 제거와 기본+딥 청소를 창 없이 자동
실행합니다. SYSTEM 권한으로 실행되지만 **모든 학생 계정 폴더**를 찾아
임시파일·브라우저 캐시까지 정리하도록 설계했습니다. 같은 버튼을 다시
누르면 예약이 해제됩니다. (⚠·잠금 항목은 자동 실행에 포함되지 않음 — 직접 확인이 필요하기 때문)

### 🔒 설치 잠금 · 재설치 차단 (강력, 되돌리기 가능)
- **재설치 차단(hosts)**: Steam·Epic·주요 VPN 배포 사이트 접속을 막아
  지운 뒤에도 다시 설치하기 어렵게 만듭니다.
- **설치 잠금**: Microsoft Store를 비활성화하고, 관리자 없이 표준 사용자가
  새 프로그램을 설치하지 못하게 정책을 적용합니다.
- **USB 자동실행 차단**(기본 켜짐, 안전): USB로 게임이 자동 실행되는 것을 막습니다.
- 모두 **"🔓 모든 잠금 해제"** 버튼 한 번으로 원상복구할 수 있습니다.

### 🛟 안전장치
- **정리 전 시스템 복원 지점 생성**(기본 켜짐): 문제가 생기면 복원 지점으로
  되돌릴 수 있습니다. (Windows 정책상 24시간에 1번만 생성됩니다)
- 정리할 때마다 `logs\` 폴더에 **감사 로그**를 남깁니다.

## 절대 건드리지 않는 것 (보호)
목록에 **아예 표시되지 않습니다**:
- 오피스 — **한컴오피스(한글/한셀/한쇼)·MS Office/365**
- **문서·그림 등 모든 개인 파일** (파일 삭제 대상에 문서/사진 폴더 미포함)
- 백신·보안 — Windows Defender·V3·알약 등
- **은행/공인인증 보안모듈** — INISAFE·CrossEX·MarkAny·AnySign·KeySharp 등
- 드라이버 — NVIDIA·AMD·Intel·Realtek 등
- 브라우저·런타임 — Chrome·Edge·Whale·Visual C++·.NET 등
- **Xbox 컨트롤러·로그인·오버레이 프레임워크**(XboxDevices·XboxIdentityProvider 등) —
  다른 프로그램이 의존할 수 있어 스토어 앱 제거에서도 항상 제외됩니다.

## 안전 원칙
- **위험한 프로그램·시스템 구성요소는 절대 자동 선택하지 않습니다.**
- 제거는 각 프로그램의 **정식 제거 관리자**로 진행합니다.
- 정리 전 **확인창**으로 전체 목록·예상 용량·영향 큰 항목(⚠)을 다시 보여줍니다.
- hosts 차단·설치 잠금은 **기본적으로 꺼져 있으며**, 켜더라도 마커/레지스트리
  값만 사용해 **"🔓 모든 잠금 해제"**로 정확히 원복됩니다.
- 삭제 중 잠긴 파일은 자동으로 건너뜁니다.

## 알아두기 / 한계
- hosts 차단은 **해당 서비스의 특정 도메인만** 막습니다 — 목록에 없는 다른
  게임/VPN은 차단되지 않고, 반대로 그 서비스를 정말 써야 하는 수업(e.g.
  Riot Games 관련 수업)이 있다면 켜지 마세요.
- 자동 정리 예약은 임시파일/캐시 위주로만 동작하며, ⚠·잠금 항목은 포함하지 않습니다.
- 딥 시스템 청소·강력 제거·잠금 적용은 **되돌리기 어렵거나 정책성 변경**이므로
  처음에는 게임/VPN 몇 개만 먼저 테스트해보길 권장합니다.
- 일부 프로그램은 조용한 제거를 지원하지 않아 자체 제거 창이 잠깐 뜰 수 있습니다.
